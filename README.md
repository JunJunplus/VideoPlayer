# VideoPlayer

#### 介绍
VideoPlayer是FFmpeg+Qt开发的播放器Demo，主要用于学习音视频，记录学习情况

#### 软件架构

已完成：

1.硬解码

2.GPU渲染视频帧

3.输出为YUV420P格式渲染

4.非精准Seek

待完成：

1.NV12格式渲染

2.精准的Seek

3.滑动条预览视频帧

4.倍速播放

5.快进、快退

6.播放列表，切换播放视频文件

7.网络流视频播放

8.动画效果、美化UI

9.qml、widget双界面，页面和逻辑分离
